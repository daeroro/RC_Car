PXA USB controllers

OHCI

Required properties:
 - compatible: Should be "marvell,pxa-ohci" for USB controllers
   used in host mode.

Optional properties:
 - "marvell,enable-port1", "marvell,enable-port2", "marvell,enable-port3"
   If present, enables the appropriate USB port of the controller.
 - "marvell,port-mode" selects the mode of the ports:
	1 = PMM_NPS_MODE
	2 = PMM_GLOBAL_MODE
	3 = PMM_PERPORT_MODE
 - "marvell,power-sense-low" - power sense pin is low-active.
 - "marvell,power-control-low" - power control pin is low-active.
 - "marvell,no-oc-protection" - disable over-current protection.
 - "marvell,oc-mode-perport" - enable per-port over-current protection.
 - "marvell,power_on_delay" Power On to Power Good time - in ms.


필수 속성 :
  - 호환 : 호스트 모드에서 사용되는 USB 컨트롤러의 경우 "marvell, pxa-ohci"여야합니다.

선택적 속성 :
  - "marvell, enable-port1", "marvell, enable-port2", "marvell, enable-port3"컨트롤러의 해당 USB 포트를 활성화합니다.
  - "marvell, port-mode"는 포트의 모드를 선택합니다.
1 = PMM_NPS_MODE
2 = PMM_GLOBAL_MODE
3 = PMM_PERPORT_MODE
  - "marvell, power-sense-low"- 전력 감지 핀이 로우 액티브.
  - "marvell, power-control-low"- 전원 제어 핀이 로우 액티브입니다.
  - "Marvell, no-oc-protection"- 과전류 보호를 비활성화하십시오.
  - "marvell, oc-mode-perport"- 포트 별 과전류 보호를 가능하게합니다.
  - "marvell, power_on_delay"전원이 켜지면 좋은 시간 - ms입니다


Example:

	usb0: ohci@4c000000 {
		compatible = "marvell,pxa-ohci", "usb-ohci";
		reg = <0x4c000000 0x100000>;
		interrupts = <18>;
		marvell,enable-port1;
		marvell,port-mode = <2>; /* PMM_GLOBAL_MODE */
	};

UDC

Required properties:
 - compatible: Should be "marvell,pxa270-udc" for USB controllers
   used in device mode.
 - reg: usb device MMIO address space
 - interrupts: single interrupt generated by the UDC IP
 - clocks: input clock of the UDC IP (see clock-bindings.txt)

Optional properties:
 - gpios:
   - gpio activated to control the USB D+ pullup (see gpio.txt)

필수 속성 :
  - 호환 : 장치 모드에서 사용되는 USB 컨트롤러의 경우 "marvell, pxa270-udc"이어야합니다.
  - reg : usb 장치 MMIO 주소 공간
  - 인터럽트 : UDC IP에 의해 생성 된 단일 인터럽트
  - clocks : UDC IP의 입력 클럭 (clock-bindings.txt 참조)

선택적 속성 :
  - gpios :
    - gpio가 USB D + 풀업을 제어하도록 활성화되었습니다 (gpio.txt 참조).


Example:

		pxa27x_udc: udc@40600000 {
			compatible = "marvell,pxa270-udc";
			reg = <0x40600000 0x10000>;
			interrupts = <11>;
			clocks = <&pxa2xx_clks 11>;
			gpios = <&gpio 22 GPIO_ACTIVE_LOW>;
		};
